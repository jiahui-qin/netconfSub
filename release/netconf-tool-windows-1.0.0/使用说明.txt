Netconf Tool - Windows发布包
=========================

注意: 此发布包尚未包含 exe 文件。

请在 Windows 环境下运行以下命令生成 exe 文件:
  cd backend
  npm run pkg:win

或者使用 scripts/create-release.bat 脚本。

开发模式启动: 双击 快速开始.bat (需要先在Windows环境安装Node.js)

文件说明:
- dist/: 前端资源文件
- README.md: 项目主文档
- docs/WINDOWS-GUIDE.md: Windows使用指南
- run-package.bat: 标准启动脚本
- 快速开始.bat: 快速启动脚本
